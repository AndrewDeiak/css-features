Flex-grow formula:

  • get the parent width: 1000
  • get the width of first and second child elements(span): 484.88 and 168.31
  • sum childs elements (span): 484.88 + 168.31 = 653.19
  • get the free space around childs elements(span): 1000px - 653.19 = 346.8
  • sum childs flex-grow properties: 1 + 1 = 2
  • free space / sum of childs flex-grow properties: 346.8 / 2 = 173.4
  • first child width: 484.88 + 173.4 = 658.28
  • second child width: 168.31 + 173.4 = 341.71
  • check if sum of childs width the same as parent: 658.28 + 341.71 = 10000
